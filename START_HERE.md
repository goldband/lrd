```

2. npm start

4. Test it out
```

## 🎮 HOW IT WORKS

### For Your Players
1. Visit your public URL
2. Sign up with email
3. See all upcoming games (Saturdays & Sundays)
4. Click: "✓ Playing", "? Maybe", or "✗ Not Playing"
5. See who else is playing
6. Done!

### For You (Admin)
1. Sign up as first user
2. Make yourself admin (see QUICK_START.md)
3. Create games with date/time/venue
4. View who's playing each game
5. Manage player accounts
